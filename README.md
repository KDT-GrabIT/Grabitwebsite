# GrabIT Landing Website

GrabIT 애플리케이션의 서비스 소개 및 다운로드 안내를 위한 공식 랜딩 웹사이트입니다.

## ✨ 주요 섹션
* **Hero Section**: 앱의 메인 컨셉 및 다운로드 링크/QR 코드 제공
* **Problem Section**: 시각장애인이 쇼핑 중 겪는 문제점 제시
* **How To Section**: GrabIT 앱의 사용 방법 및 UI 소개
* **Tech Stack Section**: 프로젝트에 사용된 기술(On-Device AI, Android 등) 소개

## 🛠 기술 스택
* **Framework**: React, Vite
* **Language**: TypeScript
* **Styling**: Tailwind CSS, shadcn/ui
* **Deployment**: Vercel

## 🚀 실행 방법
1. 의존성 패키지를 설치합니다. (npm 또는 pnpm 사용 가능)
   `npm install`
2. 개발 서버를 실행합니다.
   `npm run dev`
3. 브라우저에서 `http://localhost:5173` 으로 접속합니다.
